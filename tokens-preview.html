Cores — semânticas

Os tokens que os componentes deveriam de fato usar. Clique em "☾ / ☀ tema" no menu para alternar — esta seção inteira responde de verdade, porque usa as mesmas variáveis do site.

--color-bg — fundo da página
--color-surface — fundo de cards/chips
--color-border — bordas e divisórias
--color-text — texto principal
--color-text-secondary — corpo de texto
--color-text-muted — legendas, texto terciário
--color-accent — laranja de marca
--color-success — status "disponível", tag "solution"
--color-warning — tag "current" (comparação)
--color-inverse-bg — fundo de botão sólido

Tema atual: light — atributo data-theme na tag <html>, exatamente como no site real.

Cada card simula os dois extremos reais do clamp(): uma caixa com largura de container travada em 375px (mobile) e outra em 1440px (desktop — role a caixa se necessário). Usa container-type:inline-size + cqw em vez de vw, então o número que você vê é exatamente o que o clamp real produziria naquele viewport. Requer um navegador atual (Chrome/Safari/Firefox recentes).
