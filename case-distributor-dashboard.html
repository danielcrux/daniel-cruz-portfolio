Work / Distributor Dashboard

Giving distributors one clear view into inventory, orders and forecasts

Add a screenshot of the dashboard here

Challenge

Distributors were juggling several disconnected tools to track inventory, orders and shipments, so nobody had one reliable source of truth. Small delays in one tool cascaded into bigger ones downstream, and support tickets kept pointing back to the same root cause: too many places to check, no single answer.

Approach

I worked with engineering to consolidate the underlying data model first, then prioritized which workflows to surface based on how often distributors actually used them. The result was a modular dashboard shell that could adapt across regions without a redesign each time, backed by a small component library for filters, tables and status states so new views could ship fast and stay consistent.
